## 실행화면(과제2)
-과제2코드의실행스크린샷
<img width="1510" height="799" alt="스크린샷 2026-03-24 200846" src="https://github.com/user-attachments/assets/01e7d968-6dd5-42a9-ab3d-9c5f835ba37b" />

-과제내용
-전송이 끝나면 텍스트박스에 남겨진 메세지 삭제
-전송 후에 마우스 커서가 텍스트박스에 자동으로 위치
-마우스 클릭 대신 엔터키로도 메세지 전송 
-텍스트박스에 내용이 없을 때는 전송 비활성화

-구현내용과기능설명
-첫번째로 텍스트박스에 메세지를 입력하고 전송을 하면 자동으로 남겨진 메세지는 
clear 메서드의 연속이라고 보시면 될 것 같습니다

-두번째로 마우스 커서 자동 이동은 텍스트박스에서 focus 메서드를 이용하여 그 기능을 구현하였습니다.

-세번째로 마우스 클릭 대신 엔터키로 메세지를 전송하는 것은 텍스트박스에서 ketdown 이벤트를 만들고
그 안에 key가 enter일 때 전송이 되게 구현하였습니다.

-마지막으로 텍스트 박스에 내용이 없을 때 전송이 비활성화 되는 것은 텍스트박스에서 textchanged 이벤트를 만들고
그 안에 텍스트박스의 텍스트가 비어있을 때 전송버튼이 비활성화 되도록 구현하였습니다.

## 실행화면(과제3)
-과제3코드의실행스크린샷
<img width="1517" height="812" alt="스크린샷 2026-03-24 200939" src="https://github.com/user-attachments/assets/feb28851-a516-4d76-8fac-0ef07a8d485a" />

-과제내용
-메세지를 입력할 때 옆에 현재 시간 추가
-리스트박스에 쌓인 메세지 개수를 카운팅하여 label에 표시
-문자열 앞뒤로 지저분한 공백 제거

-구현내용과기능설명
-첫번째로 메세지를 입력할 때 옆에 현재 시간 표시는 DateTime.Now.ToString("HH:mm:ss")를 이용하여 
현재 시간을 시분초로 나타나게 하였습니다.

-두번째로 리스트박스에 쌓인 메세지 개수를 카운팅하는 것은 리스트박스의 Items.Count를 이용하여 리스트박스에 메세지가 추가될때마다
카운팅이 되도록 구현하였습니다.

-마지막으로 공백제거는 교수님께서 과제를 하시기전에 설명해주신 Trim 메서드를 이용하여 
텍스트박스에 앞뒤 공백을 제거하는 기능을 구현하였습니다.

-과제내용
-리스트박스에 특정 메세지를 클릭하고 삭제 버튼을 누르면 클릭된 메세지만 삭제
-대화 기록 삭제 버튼을 누를 시에 리스트박스에 쌓인 모든 메세지 삭제
-텍스트박스에 글자 수를 제한하고 초과시에 경고 문구 띄우기

-구현내용과기능설명
-첫번째로 리스트박스에서 특정 메세지를 클릭하고 삭제 버튼을 누르면 클릭된 메세지만 삭제하는 것은 리스트박스에서 SelectedIndex를 이용하여
그 클릭된 메세지의 인덱스를 받아서 그 인덱스에 해당되는 메세지만 삭제되도록 구현하였습니다.

-두번째로 대화 기록 삭제 버튼을 누르면 리스트박스에 모든 메세지를 삭제하는 기능은
과제2에서 텍스트박스에 남음 메세지를 삭제할 때 썼었던 clear 메서드를 리스트박스에 적용하여 간단하게 구현하였습니다.

-마지막으로 텍스트박스에 글자 수를 제한하는 것은 텍스트박스의 MaxLength 속성을 이용하여 
조건문을 접목해 <50 을 넣어서 간단하게 초과시에 간단한 경고문구를 띄우도록 구현하였습니다.
